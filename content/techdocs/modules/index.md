# 模块

下面按“功能模块”拆开讲。每一页都会尽量回答四个问题：

1. 它做什么（职责边界）
2. 入口在哪（关键文件/函数）
3. 输入/输出是什么
4. 产物落在哪里（缓存/落盘）

# FOMC101（学习模式）

## 命名说明

前端展示名称为 **FOMC101**，技术层保留 `fed101`：

- 路由：`/fed101`
- API：`/api/fed101/*`
- 内容目录：`content/fed101/`

## 内容从哪里来

- 内容目录：`content/fed101/`
- 读取与渲染：`src/fomc/apps/web/fed101.py`

每个章节是一个 Markdown 文件，顶部用 frontmatter 描述元信息：

- `slug/title/order/summary`
- `hidden` 用于隐藏旧页面但保留兼容访问

## cell 是怎么运行的

FOMC101 正文里可以插入 `fomc-cell` 代码块（JSON）。后端会把它替换为占位 DOM，前端再渲染成可运行小组件。

- 后端解析/运行：`src/fomc/apps/web/fed101.py`
- 执行 API：`POST /api/fed101/cell`
- 前端渲染：`src/fomc/apps/web/static/fed101.js`

常见 cell 类型：

- `indicator_chart`
- `labor_figure` / `cpi_figure`
- `taylor_model`
- `meeting_statement_diff`
- `meeting_decision_brief`

## meeting_id 的意义

学习模式里你可以选择一个 `meeting_id` 作为示例会议：

- 对齐研报月份窗口（NFP/CPI）
- 对齐规则模型的结束日期

上下文来自：`src/fomc/apps/web/backend.py:get_meeting_context`

# Fed101（学习模式）

## 内容从哪里来

- 内容目录：`content/fed101/`
- 读取与渲染：`src/fomc/apps/web/fed101.py`

每个章节是一个 Markdown 文件，顶部用 frontmatter 描述元信息：

- `slug/title/order/summary`
- `hidden` 用于隐藏旧页面但保留兼容访问

## cell 是怎么运行的

Fed101 的正文里可以插入 `fomc-cell` 代码块（JSON）。后端会把它替换成占位 DOM，前端再把它渲染成可运行的小组件。

- 后端：`src/fomc/apps/web/fed101.py`（解析、运行 cell）
- API：`POST /api/fed101/cell`（见 `src/fomc/apps/web/main.py`）
- 前端渲染：`src/fomc/apps/web/static/fed101.js`

## meeting_id 为什么重要

学习模式里你可以选择一个 `meeting_id` 作为示例会议。很多 cell 会用它做“对齐”：

- 自动推断研报月份（例如 NFP/CPI 的月度窗口）
- 对齐规则模型的结束日期到会议决议日

上下文来自：`src/fomc/apps/web/backend.py:get_meeting_context`
